# TypeScript

In this section, we will use `TypeScript`, the super set of JavaScript!
For that, let's install TypeScript globally like this: `npm install -g typescript`

## TypeScript commands

- `tsc file_name.ts` - this will convert a typescript file to javascript
- `tsc --init` - creates **tsconfig.json**
